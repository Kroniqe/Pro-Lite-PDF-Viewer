Pro-Lite PDF Viewer

A lightweight, high-performance PDF viewer for Windows built with Python. Designed as a "bloat-free" alternative to heavy commercial readers, featuring tabbed browsing, annotation tools, and instant startup times.

Installation

Option 1: Download .EXE

Check the Releases page for the standalone Windows executable. No installation required‚Äîjust download and run.

Option 2: Run from Source

Clone this repository.

Install dependencies:

pip install -r requirements.txt


Run the application:

python pdf_viewer.py


Build it Yourself

To create the standalone .exe file:

Install PyInstaller: pip install pyinstaller

Run the build command:

pyinstaller --noconsole --onefile --icon=app_icon.ico pdf_viewer.py


The executable will appear in the dist folder.
